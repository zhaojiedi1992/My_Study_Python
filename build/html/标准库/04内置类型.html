内置类型¶

真值测试¶

下面几类值为False

  • 定义为None或者为False
  • 0值，如0，Decimal(0),Fraction(0,1)
  • 空的序列或者集合，如‘’，(),[],{},set(),range(0)

boolean操作符¶

操作符 结果 注记
x or y if x is false,then y, else x 短路操作，如果x为true，后续不判断了
x and y if x is false , then x else y 如果x是false了，y就不用在判断了
not x if is false,then true,else false  

比较操作符号¶

操作符 描述，”对应类方法”  
< 小于 __lt__()
<= 小于等于 __le__()
> 大于 __gt__()
>= 大于等于 __ge__()
== 等于 __eq__()
!= 不等于 __ne__()
is 对象识别 无
is not 反向的对象识别 无
